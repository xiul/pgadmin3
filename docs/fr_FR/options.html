Options de pgAdmin

pgAdmin a un ensemble d'options de configuration pour vous aider à améliorer pgAdmin. Si vous pensez qu'une fonctionnalité supplémentaire devrait être optionnelle ou configurable, merci d'envoyer aux développeurs un courriel sur pgadmin-hackers@postgresql.org.

Habituellement, pgAdmin se configure à partir de l'application. Le dialogue des Options est disponible dans Fichier -> Préférences.
