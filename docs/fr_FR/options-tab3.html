Options de pgAdmin - Onglet 3 (Navigateur)

Afficher les objets système dans le navigateur - cette option permet aux objets tels que les schémas système (comme pg_temp*) et les colonnes système (comme xmin et ctid) d'être inclus ou exclus du navigateur.

La liste de cases à cocher est utilisée pour sélectionner les types d'objets à afficher dans le navigateur. Par défaut, seuls les types d'objet les plus fréquents sont affichés. Réduire le nombre des types d'objets peut augmenter la rapidité de pgAdmin lors du requêtage des catalogues système. Utilisez le bouton Par défaut pour réinitialiser la liste à sa configuration par défaut.
