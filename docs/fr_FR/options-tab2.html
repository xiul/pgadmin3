Options de pgAdmin - Onglet 2 (Préférences)

  • Police - Cette option vous permet de sélectionner la police utilisée en standard dans pgAdmin. Cela peut se révéler utile dans les pays où le jeu de caractères multi-octets ne s'affichent pas bien avec les polices standards ou si vous utilisez une police différente de celle configurée sur votre système.
     
  • Lire et écrire des fichiers Unicode UTF-8 - Sélectionnez cette option pour activer l'écriture de fichiers Unicode UTF-8 par défaut dans pgAdmin.
     
  • Ne pas demander pour les fichiers modifiés en sortie - Sélectionner cette option empêchera pgAdmin de vous proposer de sauvegarder les requêtes SQL lorsque vous quitterez l'application.
     
  • Confirmer la suppression d'objet ? - Sélectionnez cette option pour vous assurer que pgAdmin vous donne une chance de confirmer avant de réellement supprimer les objets (recommendé).
     
  • Afficher les utilisateurs pour les droits ? - Lors de l'accès à l'onglet Droits sur les objets qui l'affichent, sélectionner cette option incluera les noms des utilisateurs ainsi que les groupes dans la liste en question. Désactiver cette option est utile sur de gros systèmes avec de nombreux utilisateurs organisés en groupes.
     
  • Afficher les propriétés d'un objet sur un doubel clic dans le navigateur ? - Sélectionner cette option autoriser le double-clic dans le navigateur pour ouvrir le dialogue de propriétés des objets pour cet objet. Certains utilisateurs préfèrent désactiver cette option pour utiliser le doublic clic pour naviguer.
     
  • Ne pas afficher les astuces de gouru, j'en suis un - pgAdmin III affiche des fenêtres d'astuces de temps en temps s'il pense que vous pourriez en avoir besoin. Si cela vous ennuie et que vous voulez supprimer ce comportement, vous pouvez cocher cette case.
     
  • Réinitialiser les astuces de gourous - Quand une astuce de gourou est affiché, vous pouvez supprimer son affichage futur. Si vous voulez que l'astuce ré-apparaisse, cochez cette case.
