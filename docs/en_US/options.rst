***************
pgAdmin Options
***************

pgAdmin has a selection of configuration options to help it work
as you prefer. If you think of an additional feature that should be
optional or configurable though, please send the developers an email
at `pgadmin-hackers@postgresql.org <mailto:pgadmin-hackers@postgresql.org>`_.

Normally pgAdmin will be configured from within the application.
The Options dialogue can be found under *File -> Options*.

For help on each of the individual tabs, please click the
appropriate link below.
